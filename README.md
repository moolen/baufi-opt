# Baufinanzierungs-Optimierer

A web application for calculating and optimizing mortgage loan repayment schedules.

## What It Does

Create and manage multiple loans, simulate special payments (Sondertilgungen), and visualize detailed amortization schedules with interest impact analysis. Compare different repayment strategies to find the most efficient path to paying off your mortgage.

## Quick Start

Navigate to the `ui` directory and run:

```bash
cd ui
npm install    # Install dependencies
npm run dev    # Start development server at http://localhost:3000
```

To build for production:

```bash
npm run build  # Generate optimized build
npm run preview # Preview the built app
```

## Tech Stack

- **React 19** + TypeScript
- **Vite** for bundling
- **Tailwind CSS** for styling
- **Recharts** for visualizations
- **Lucide React** for icons
